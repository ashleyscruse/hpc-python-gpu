# HPC Python Lab

Running Python on HPC using Jupyter via TACC.

## Guide

**[Setup guide](https://morehouse-supercomputing.github.io/hpc-python-gpu/)** — how to launch Jupyter on a TACC compute node from the terminal or the web portal.

## Notebook

`notebooks/jupyter_on_hpc.ipynb` — matrix multiplication, multiprocessing, and large data on HPC.

## Quick Start (on TACC)

```bash
cd $WORK
git clone https://github.com/morehouse-supercomputing/hpc-python-gpu.git
cd hpc-python-gpu
```

Then open the notebook in Jupyter.
